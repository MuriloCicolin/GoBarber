# Go barber

Go Barber é um aplicativo de barbearia onde é possível criar, atualizar, cancelar agendamentos. 
 Backend feito em Node Js com as seguintes ferramentas:

- Sequelize
- Nodemon
- Docker
- EsLint 
- Prettier
- Mongoose
- Express
- Multer
- Nodemailer
- Date-Fns
- Json Web Tokens
- Redis
- Queue
- Youch
- Yup
- Handlebars
